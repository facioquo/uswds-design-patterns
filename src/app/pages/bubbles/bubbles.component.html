Floating bubbles

A floating collection of bubble-shaped cards without use of the U.S. Web Design System Layout grid. Learn more about this design pattern at the bottom of this page.

{{ card.title }}

About this pattern

Use this design pattern when you want consistent spacing between fixed size [bubble] cards and symmetrical centering of 3-6 objects. It'd be difficult to do this using the standard USWDS Layout grid.
