About this pattern

Side navigation

Use this design pattern when you want consistent spacing between fixed size [bubble] cards for different screen widths, where the USWDS Layout grid doesn't work. This can be done with complex use of USWDS utilities; however, use of standard CSS flex properties is simpler.

What we changed

This isn't a modification of a USWDS component, but it does adhere to its broad guidelines and styling.
